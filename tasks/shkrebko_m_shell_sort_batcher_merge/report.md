# 1. Введение
Нужно реализовать параллельную сортировку массива целых чисел с использованием алгоритма сортировки Шелла и четно-нечетного слияния Бэтчера. Цель заключается в реализации последовательной и параллельной версий алгоритма, а также в их сравнении по производительности.

# 2. Постановка задачи
Дан массив целых чисел произвольного размера. Требуется отсортировать его по возрастанию.

Вход:  вектор целых чисел произвольной длины
Выход: вектор целых чисел, отсортированный по возрастанию
# 3. Описание алгоритма(Sequential)
Алгоритм состоит из 1 этапа:

1 Сортировка Шелла - сортировка с убывающим шагом, начиная с n/2, где n - размер массива.

# 4. Описание алгоритма (MPI)
Распределение данных:
Исходный массив делится на части примерно одинакового размера между процессами.

Шаблон коммуникаций:

MPI_Bcast для передачи размера массива

MPI_Scatterv для отправки каждому процессу его части массива

MPI_Send/MPI_Recv для попарного слияния отсортированных частей

MPI_Bcast для рассылки отсортированного массива от корневого процесса всем процессам

Роли процессов:

Все процессы сортируют свою часть массива с помощью сортировки Шелла

Процессы объединяются в пары для слияния с помощью сети Бэтчера

Процесс с рангом 0 собирает финальный отсортированный массив
