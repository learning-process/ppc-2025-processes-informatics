- **Вариант:** сортировка массива  

---

## 1. Введение

Цель работы — реализовать алгоритм сортировки пузырьком (вариант чёт–нечётной перестановки) и распараллелить его при помощи MPI.

---

## 2. Постановка задачи

Задан массив целых чисел:

```
A = [a₀, a₁, …, aₙ₋₁]
```

Необходимо реализовать последовательную и MPI версии алгоритма.

---

## 3. Последовательная версия (SEQ)

Используется алгоритм чёт–нечётной перестановки.

---

## 4. Параллельная версия (MPI)

Распределение данных — `MPI_Scatterv`, локальная сортировка `std::sort`, сбор — `MPI_Gatherv`, финальная сортировка на процессе 0.
