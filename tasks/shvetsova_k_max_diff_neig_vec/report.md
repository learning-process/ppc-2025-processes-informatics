## Описание

- **Задача**: Нахождение наиболее отличающихся по значению соседних элементов вектора
- **Вариант**: 8
- **Технология**: SEQ, MPI
- **Описание** 
Была решена задача о нахождении наиболее отличающихся по значению соседних элементов вектора. Были разработаны два алгоритма для решения этой задачи: SEQ и MPI. Также были разработаны тесты для проверки корректности работы обоих алгоритмов и оценки их производительности. В MPI версии алгоритм разделяет данные между процессами, находит локальные максимумы на каждом из процессов, обрабатывает граничные значения каждого процесса и затем вычисляет итоговый максимум. Оба алгоритмы успешно проходят тесты.
